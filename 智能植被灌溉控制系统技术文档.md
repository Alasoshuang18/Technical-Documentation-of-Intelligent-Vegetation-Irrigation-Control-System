# 智能植被灌溉控制系统技术文档

## 1. 项目概述

本系统基于 STM32（兼容 Arduino 框架）开发，是一款集成了环境感知、多模式切换、人机交互与预警功能的智能灌溉解决方案。系统通过采集 4 路土壤湿度传感器的平均值，结合实时逻辑判断，实现对水泵（继电器控制）的自动化调度。

## 特殊配置说明

- **JTAG 禁用**：代码通过 `AFIO->MAPR` 寄存器禁用了 JTAG 调试功能，从而释放 **PB3** 和 **PB4** 引脚作为普通 GPIO，确保矩阵键盘的完整扫描。

## 2. 系统状态机架构

系统逻辑运行在三个互斥的状态之间，确保过程的安全可控：

1. **空闲状态 (free_state)**：
   - 实时监测土壤湿度。
   - 待机状态，继电器与蜂鸣器均处于关闭位。
   - 持续检查触发条件（湿度或定时）。
2. **预警状态 (WARNING)**：
   - 当触发条件满足时进入此状态，持续 **60 秒**。
   - 蜂鸣器以 500ms 为周期闪烁鸣叫，提醒操作人员设备即将喷水。
3. **灌溉状态 (WATERING)**：
   - 继电器闭合，水泵启动。
   - 系统根据当前模式判断停止时机（时间达标或湿度回升）。

## 3 数据采集与精度控制

- **多点采样**：系统同时读取 4 路传感器 ADC 值，通过 `map` 函数将其映射为 0-100% 的湿度百分比。
- **均值滤波**：取 4 路数据的算术平均值作为判断依据，有效降低了单点故障或传感器误差导致的误触发。
- **滞回比较**：在模式 1 下，停止阈值设为 `阈值 + 5%`。这种设计防止了数值波动导致继电器频繁“打火”开关，延长硬件寿命。

## 3.1 智能混合逻辑（模式 3）

在模式 3 下，系统表现出极高的智能化：

- 如果定时时间到了但湿度仍充足，依然会触发灌溉流程。
- 如果灌溉时长已到但湿度仍未达标，系统会自动重置时长计数器，继续灌溉直至土壤湿润，确保植物获得足够水分。

## 3.2 紧急干预机制

- **一键启停**：无论系统处于何种模式，按下键盘 **'left'** 键立即可强行停止灌溉；按下 **'right'** 键可直接跳过触发判断进入预警喷水流程。

## 3.3 运行模式详解

系统提供四种灵活的工作模式，可通过键盘 **'F2'** 键循环切换：

| **模式编号** | **模式名称**   | **触发逻辑**               | **停止逻辑**                   |
| ------------ | -------------- | -------------------------- | ------------------------------ |
| **Mode 1**   | **纯湿度模式** | 平均湿度 < 设定阈值        | 平均湿度 >   (阈值 + 5%)       |
| **Mode 2**   | **定时模式**   | 间隔达 24 小时自动触发     | 喷水时长达到设定分钟数         |
| **Mode 3**   | **混合模式**   | 湿度不足 **或** 定时到达   | 喷水时间达标 **且** 湿度已恢复 |
| **Mode 4**   | **强制停止**   | 系统锁定，不执行自动化动作 | 全局关闭状态                   |















## 4. 硬件资源配置

## 4.1 引脚定义表

| **组件名称**        | **引脚编号**           | **功能描述**                             |
| ------------------- | ---------------------- | ---------------------------------------- |
| **主控芯片**        | STM32F103 系列         | 系统核心运算与控制                       |
| **蜂鸣器 (Buzzer)** | **PB0**                | 预警状态下进行声光提示（500ms 翻转鸣叫） |
| **继电器 (Relay)**  | **PA8**                | 控制外部水泵或电磁阀驱动                 |
| **土壤湿度传感器**  | **PA1, PA2, PA3, PA4** | 4 路模拟信号采集，支持多点监测           |
| **OLED 显示屏**     | **I2C (0x3C)**         | 128x64 分辨率，显示实时数据与菜单        |
| **矩阵键盘**        | **PB3,4,8,9,11-15**    | 5x4 布局，用于模式切换与参数设置         |



## 5. 人机交互指南

## 5.1 实时监控界面 (Monitor Mode)

- **MOD**: 显示当前运行模式。
- **LIM/T**: 显示当前的湿度阈值上限和预设喷水时长。
- **AVG**: 实时显示当前 4 路传感器的平均湿度。
- **ST**: 状态缩写（free-空闲，WARN-预警，WTR-灌溉）。
- **S1-S4**: 底部展示 4 个探测点各自的具体湿度数值。
- **F2:**切换四种模式
- **F1:**设置阈值和时间(定时器触发的时长)

## 5.2 参数设置界面 (Setting Mode)

| 按键    | 功能         | 说明                               |
| :------ | :----------- | :--------------------------------- |
| **F1**  | 进入参数设置 | 进入设置模式，可修改湿度和时长参数 |
| **F2**  | 切换工作模式 | 循环切换系统工作模式               |
| **->**  | 一键喷水     | 手动启动灌溉流程                   |
| **<-**  | 一键停水     | 立即停止所有灌溉操作               |
| **Ent** | 确认键       | 在设置模式中确认输入               |
| **Esc** | 退出键       | 退出设置模式                       |
| *****   | 未使用       | 预留功能键                         |
| **#**   | 未使用       | 预留功能键                         |

注意:**一键喷水**运行基于当前模式下
